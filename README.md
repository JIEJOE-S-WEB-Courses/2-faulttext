## **Describe📋️**
以CSS的clippath裁剪路径加上mix-blend-mode混合模式以及transform位移，共同构成的故障文字动画效果

<br>

## **Notice⚠️**
- 混合模式的多种类型样式详见[MDN](https://developer.mozilla.org/zh-CN/docs/Web/CSS/mix-blend-mode)
- 效果主体不一定是文字，也可以是盒子元素或者图片，但是效果逻辑都是一样的
